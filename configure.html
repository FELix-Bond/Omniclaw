🗂️ Google Full Suite — Drive, Docs, Sheets, Calendar
One-time setup (~10 min): Creates a Google Cloud project that gives your agents access to Drive, Docs, Sheets, and Calendar. Setup runs automatically on first install — you just need to click "Allow" in your browser once.

Step-by-step:
1. Go to console.cloud.google.com → Create a new project (name it "OmniClaw")
2. APIs & Services → Enable APIs → enable: Gmail API, Drive API, Docs API, Sheets API, Calendar API
3. APIs & Services → Credentials → Create Credentials → OAuth 2.0 Client ID → Application type: Desktop app
4. Copy the Client ID and Client Secret below
5. On first install, your browser will open once for you to click "Allow" — then it's fully automated forever
